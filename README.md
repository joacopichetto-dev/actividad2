📌Descripcion

Este proyecto corresponde a la Actividad 2 de Python 2026, donde se resuelven distintos problemas utilizando estructuras de datos, procesamiento de texto, validaciones y simulaciones.

-Resolver problemas más complejos con estructuras de datos.

-Separar lógica y ejecución (src + notebooks).

-Trabajar con datos reales e inconsistentes.

🛠️ Tecnologías utilizadas

-Lenguaje: Python 3

-Entorno: Jupyter Notebook

-Librerías estándar (random, re, etc.)-

## ⚙️ Instalación

1. Cloná el repositorio:

```bash
git clone https://github.com/tu-usuario/tu-repo.git
cd tu-repo
```

2. Creá un entorno virtual:

```bash
python -m venv .venv
```

3. Activá el entorno virtual:

**Windows:**

```bash
.venv\Scripts\activate
```

**Mac / Linux:**

```bash
source .venv/bin/activate
```

4. Instalá las dependencias:

```bash
pip install -r requirements.txt
```
## 🚀 Ejecución

1. Con el entorno virtual activado, iniciá Jupyter:

```bash
jupyter lab
```

2. Abrí la carpeta `notebooks/` y ejecutá los archivos `.ipynb`.
